\Premise_Tabs

Premise Tabs Class

This class is supposed to do one thing well - output markup for tabs. That means that this class is meant to provide developers with the markup necessary for creating tabs, but should let the developer control functionality and styles. Of course, by default this class outputs working tabs with basic styles and fucntionality, but the idea is that they can be easily overridden or customized.
